
# Graphs Folder

This folder contains all the graphical visualizations generated during the **Cybersecurity Salary Analysis** project.

---

## Contents

| Filename              | Description                                  |
|-----------------------|----------------------------------------------|
| 1.png  | Histogram showing overall salary distribution. |
| 2.png | Bar chart of average salary by job title (top 10). |
| 3.png | Boxplot of salary spread by experience level. |
| 4.png  | Scatter plot showing remote work ratio vs salary. |

---

## Usage

- These images are referenced in the main project report and Jupyter notebook.  
- You can download or view these graphs to better understand the salary trends in cybersecurity.  
- The graphs are saved in **PNG** format with high resolution suitable for reports and presentations.

---

## How to Generate

The graphs were generated using **Matplotlib** in Google Colab.  
To regenerate or customize graphs, run the analysis notebook:

`Cyber_Salary_Analysis.ipynb`
